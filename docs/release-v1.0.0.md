# v1.0.0 — Trust-Aware Metadata Reasoning Prototype

## Highlights

- deterministic planning before LLM SQL generation
- governance-aware orchestration across PII, RBAC, and scan-risk checks
- confidence propagation across retrieval, join reasoning, governance, completeness, and intent clarity
- honest refusal with explicit failure taxonomy
- evaluation harness with grouped benchmark fixtures and measurable trust behavior
- single-page Streamlit reasoning demo for SQL generation, refusal, and governance blocking
